## Code of Conduct ##

Our Commitment

We are committed to creating a respectful, inclusive, and professional community.
All participants are expected to contribute in a positive and constructive manner.

Expected Behavior
	•	Be respectful and considerate in all interactions
	•	Provide constructive feedback
	•	Accept feedback gracefully
	•	Focus on collaboration and shared goals
	•	Maintain professionalism in discussions and contributions

Unacceptable Behavior
	•	Harassment, discrimination, or abusive language
	•	Personal attacks, trolling, or hostile behavior
	•	Publishing private information without permission
	•	Any conduct that would be inappropriate in a professional environment

Scope

This Code of Conduct applies to all project spaces, including:
	•	Issues
	•	Pull requests
	•	Discussions
	•	Comments
	•	Documentation and related communication

Enforcement

Project maintainers may remove or modify content that violates this Code of Conduct.

Repeated or serious violations may result in temporary or permanent removal from participation in the project.

By participating in this repository, you agree to abide by this Code of Conduct.
